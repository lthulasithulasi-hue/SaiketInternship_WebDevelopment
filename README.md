## About
This repository contains the exercises, tasks, mini-projects, and assignments completed as part of the Saiket web development internship. Each project/demo is intended to demonstrate concepts learned (HTML, CSS, JavaScript, basic front-end tooling, and small full-stack examples where applicable).

## Projects & Assignments
Each top-level folder generally represents a single task or project. Example structure (actual folders may vary):
- project-01-responsive-layout/ — responsive layout practice
- project-02-js-interactivity/ — DOM manipulation & events
- project-03-api-integration/ — fetching data from APIs
- project-04-form-validation/ — client-side form validation
- project-05-node-express/ — small Express server (if present)
- assignments/ — collection of smaller exercises and notes

Open each project folder for a dedicated README (recommended) describing that task's purpose and run instructions.

## Tech Stack & Languages
- HTML5
- CSS3 (Flexbox, Grid; may include preprocessors in some projects)
- JavaScript (ES6+)
- Optional: Node.js / Express for backend tasks or simple APIs
- Development tools: npm/yarn, local dev server, VS Code (Live Server) etc.

## How to Run Each Project
Because this repo contains multiple independent projects, check inside the folder of the project you want to run. Recommended per-project README content:
- Purpose / brief description
- Prerequisites (node version, env vars)
- Install steps (e.g., `npm install`)
- Run / development command (e.g., `npm start`)
- Build / production command (if applicable)
- Demo link (if deployed)
